Pour installer l'application, vous devez suivre les étapes suivantes :

Renommez le fichier config.exemple.php en config.php et remplissez-le avec vos identifiants et configurations spécifiques.

Ouvrez votre terminal et exécutez la commande "composer dump-autoload"

Cela permettra d'installer le dossier vendor contenant les bibliothèques et les composants nécessaires à l'application.

Une fois ces étapes terminées, l'application sera prête à être utilisée.
